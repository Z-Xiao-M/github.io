# 前言
没错，今天是我的生日，有一说一，时间过得还真是蛮快。转眼我就27岁了。

# 关于小满
其实我一向对生日没有多少的关注，我差不多也是这个礼拜二的时候，才突然意识到周六是我的生日。甚至我整个上半年都没太关注过节假日什么的，差不多都是等到了临近的时候，有同事问我假期有什么安排的时候，才晓得快要放假了。（只能说是太“充实”了）。也有朋友在今年小满这个节气的时候，给我送了生日祝福，非常的感谢。今年的小满（节气）也是5月21日，有朋友说小满这个节气还怪好的，卡在这个时间点。其实还真挺巧，我出生的那天，是99年的5月21日，也刚好是小满， 这也是我名字由来了。对于生日，我们那边选择的是农历（不跟着节气走），所以大部分的时候生日都是不固定的，偶尔会刚好撞见在小满当天。所以很多朋友会感到困惑，不过我无所谓，不用在意这些细节，反正送祝福我就厚脸皮收着，也没啥大不了的，对吧~

拿手机瞄了一眼，我大概在这个世界，物理存在了九千八百多天了。四舍五入就是一万天了，依稀记得有一句话是说人生短短几万天来着，这么一想时间这个货币还是太稀缺了，而且不提供`充值`渠道~~~ 所以怎么把剩余的时间花的更有`价值`，对于低精力（懒鬼）的我而言，还真是一件非常难的事情呢！要是能够自己选择完美时间线就好了 （来自懒鬼的吐槽）。

# 关于27
回到现实，虽然我对于生日没有多少感慨，但是对27这个数字还是有点感慨的，但是我又不能很精确的描述出这个感觉。不知道大家能不能理解，就这个数字它处于26和28之间，我本能的感觉26还挺年轻，但是28我就觉得好大（当然可能也只是对我而言，也有可能就只是单纯的意识到自己老了）。勉强给出一个安慰自己的理由的话，就是至少目前看起来，我还没有多少登味。其实细想下来，我似乎好像一直都是相对淡淡的，年轻的时候也没有多少少年心气的样子，真是不知道该哭还是该笑了。

# 关于记忆
偶尔会回想一些过去的事情，然后突然惊觉，原来那些事情过去了这么久了。但是并不是像这一段话描述的一样。

> “年轻的时候，我捡了一把枪，因为好玩，我开了一枪，没有谁受伤。多年之后，正在走路的自己，听到风声，我一回头，子弹正中眉心。”

我可写不出那么有文采的东西，忘了这话是史铁生还是谁写的来着，无所谓无所谓。我更多的时候其实就是想起一些流水账罢了，碰到朋友的时候，还会和他们聊起这些事情，有的朋友能够会回忆起，一起笑哈哈，一起补充细节，也有的朋友说“还有这事？我咋不记得了”。怎么说呢，时间过的真是太快了，带来的变化也确实是非常的大，我的同龄人中有的成了家，有的回去继承家产了，有的已经走了（唉~）。依稀记得去年还是前年的春节的时候，和几个老朋友在老家聚餐吃饭的时候，有个朋友是高中老师，当时的她说她在担任27届学生的班主任还是科任老师来着，突然想起来我们几个就是17届的学生，当时忍不住感慨：没想到咱们几个朋友已经相识十多年了。现在想起来也确实，毕竟我出来工作也有了五年了，五年了也就两份工作，一份干了一年，一份一直到现在，好在走的弯路不算太多，特别是现在的工作是我喜欢的工作，很感谢我的师父指导我入行以及对我的信任，翻了一下提交记录，差不多也有600多次了。
